## Video Commander

A desktop application for video engineers: inspect, analyze, convert, and package media with a modern UI powered by FFmpeg and specialized tooling.

- **Inspect** containers, codecs, and metadata
- **Analyze** DASH / HLS manifests
- **Measure** quality with VMAF and other metrics
- **Transcode** with configurable pipelines
- **Package & deliver** for streaming
- **Run** jobs locally or in the cloud
