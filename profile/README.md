# Video Commander

**Professional video engineering tools.**

Video Commander builds modern tools for inspecting, processing, and delivering video.  
Our mission is to make advanced video workflows accessible to developers, engineers, and creators.

We focus on **high-performance tooling around FFmpeg, MP4, streaming formats, and video analysis.**

---

## 🚀 Projects

### Video Commander
A desktop application for video engineers.

Inspect, analyze, convert, and package media files with a modern UI powered by FFmpeg and specialized tooling.

Features include:

- Media inspection (containers, codecs, metadata)
- DASH / HLS manifest analysis
- VMAF and quality metrics
- Transcoding pipelines
- Packaging and delivery workflows
- Local and cloud job execution
